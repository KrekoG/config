# Config

scripts to make migrating my system easier.

## install packages

- `sudo apt-get install sudo tilda git vim`

## tilda

- `cp ./software/tilda/config_0 ~/.config/tilda/config_0`

## ssh

- copy keys to `~/.ssh/`
- `chmod 600 ~/.ssh/*`

## git

- `cp ./software/git/.gitconfig ~/.gitconfig`
